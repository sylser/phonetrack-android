Το PhoneTrack είναι μία εφαρμογή συνεχούς καταγραφής συντεταγμένων τοποθεσίας.
Η εφαρμογή λειτουργεί στο παρασκήνιο. Τα σημεία αποθηκεύονται στην επιλεγμένη συχνότητα και
ανεβαίνουν στο διακομιστή ζωντανά. Αυτή η εφαρμογή καταγραφής λειτουργεί με
[https://gitlab.com/eneiluj/phonetrack-oc Εφαρμογή PhoneTrack Nextcloud] ή οποιοδήποτε
προσαρμοσμένος διακομιστής (GET ή POST HTTP αιτήματα). Η εφαρμογή PhoneTrack μπορεί επίσης απομακρυσμένα
να χειριστεί με εντολές SMS.


# Δυνατότητες

- Καταγραφή σε πολλούς προορισμούς με πολλαπλές ρυθμίσεις (συχνότητα, ελάχιστη απόσταση, ελάχιστη ακρίβεια, σημαντική κίνηση)
- Καταγραφή στην εφαρμογή PhoneTrack Nextcloud (εργασία καταγραφής PhoneTrack)
- Καταγραφή σε οποιονδήποτε διακομιστή που μπορεί να λάβει αιτήματα HTTP GET ή POST (προσαρμοσμένη εργασία καταγραφής)
- Αποθήκευση θέσεων όταν το δίκτυο δεν είναι διαθέσιμο
- Απομακρυσμένος έλεγχος μέσω SMS:
    - λήψη θέσης
    - ενεργοποίηση συναγερμού
    - εκκίνηση όλων των εργασιών καταγραφής
    - παύση όλων των εργασιών καταγραφής
    - δημιουργία νέας εργασίας καταγραφής
- Εκκίνηση κατά την εκκίνηση του συστήματος
- Εμφάνιση συσκευών στο χάρτη μίας συνεδρίας του Nextcloud PhoneTrack
- Σκοτεινό θέμα
- Πολυγλωσσικό περιβάλλον χρήστη (μεταφρασμένο στο https://crowdin.com/project/phonetrack )

# Απαιτήσεις

Αν θέλετε να χρησιμοποιήσετε την εφαρμογή του Nextcloud PhoneTrack θα χρειαστείτε:

- ένα server με εγκατεστημένο το Nextcloud
- την εφαρμογή Nextcloud Phonetrack ενεργοποιημένη σε αυτόν

Διαφορετικά, δεν υπάρχουν άλλες απαιτήσεις! (Εκτός από το Android > = 4.1)
